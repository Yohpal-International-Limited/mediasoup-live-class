# mediasoup-live-class

A live class / streaming platform built on [mediasoup](https://mediasoup.org) — a WebRTC SFU (Selective Forwarding Unit).

## Features

- Real-time audio/video conferencing via mediasoup
- Screen sharing with full UI support
- Persistent chat system with message ACK protocol, status tracking, and retry
- Host system — first peer in a room is automatically designated as host
- Participant list with host badge
- Video grid layout with overflow scrolling
- WebRTC stats and media info panels

## Custom Extensions (share branch)

| Feature | Description |
| ------- | ----------- |
| Chat | Full messaging with delivery ACKs, status indicators, and persistence |
| Host Role | First peer to join a room becomes host (visible in sidebar) |
| Screen Share UX | Proper error handling and cancel detection |
| Display Name Sync | Redux `me.displayName` synced with landing page input |
| Video Grid | Overflow scrolling with consistent tile sizing |

## Prerequisites

- Node.js >= 22
- [mediasoup build dependencies](https://mediasoup.org/documentation/v3/mediasoup/installation/#requirements)

## Installation

```sh
git clone https://github.com/kevwasonga/mediasoup-live-class.git
cd mediasoup-live-class
```

### Server

```sh
cd server
npm install
# See server/README.md for configuration
npm start
```

### App

```sh
cd app
npm install --legacy-peer-deps
npm start
```

## Configuration

Query parameters can be passed in the URL:

| Parameter       | Type    | Description                | Default      |
| --------------- | ------- | -------------------------- | ------------ |
| `roomId`        | String  | Room ID                    | Autogenerated |
| `displayName`   | String  | Participant display name   | Autogenerated |
| `mic`           | Boolean | Enable microphone by default | `true`     |
| `webcam`        | Boolean | Enable webcam by default   | `true`       |
| `forceTcp`      | Boolean | Force TCP for media        | `false`      |
| `forceVP8`      | Boolean | Force VP8 codec            | `false`      |
| `forceH264`     | Boolean | Force H264 codec           | `false`      |
| `forceVP9`      | Boolean | Force VP9 codec            | `false`      |
| `forceAV1`      | Boolean | Force AV1 codec            | `false`      |
| `stats`         | Boolean | Show WebRTC stats panel    | `false`      |
| `info`          | Boolean | Show media info panel      | `false`      |

See the [original mediasoup-demo docs](https://github.com/versatica/mediasoup-demo) for the full list.
